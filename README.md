# CSC4005 - Lab 0 Starter Kit

## Muc tieu

## Yeu cau
- Python 3.10
- pip

## Huong dan chay
1. conda create -n csc4005-dl python=3.10
2. conda activate csc4005-dl
3. pip install -r requirements.txt
4. python check_env.py
5. python run_smoke_test.py
